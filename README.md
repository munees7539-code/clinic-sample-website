# clinic-sample-website

🏥 Zenith Medical Centre
-----
Advanced Precision Healthcare for the Modern Age
          A modern, high-end clinic portfolio website designed to deliver a premium healthcare experience online.
          Built using HTML, CSS, and JavaScript, this project showcases a medical center with specialties, expert doctors, and a seamless patient interaction system.

🌐 Live Preview
---
👉 https://clinic-sample-website-six.vercel.app/

🧬 Overview
----
Zenith Medical Centre is a design-focused healthcare website that blends clinical excellence with a patient-first digital experience.
It transforms a traditional clinic presence into a modern, interactive, and responsive web platform.

✨ Key Features
---
🩺 Hero Section – First Impression

A bold and elegant landing section highlighting:

Clinic identity
Certification (ISO standard)
Core message: “Medicine for the Modern Age”
Quick stats (Specialists, Wait-Time Policy)
